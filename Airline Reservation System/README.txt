Airline reservation system version 1.0, Last update: 07.01.2014

General guide:
----------------------------------------------------------------------------------------------------------------------------------------
																       	|
- This CD contains folders:														|
																	|
	1. doc:     contains documentation for the project 										|
	2. exe:     contains executable program ready for use and library "libgcc_s_dw2-1.dll" which is needed for execution		|
	3. project: contains whole project with all source files, .c, .h, .o, .cpb etc.							|
	4. src:     contains all source files .c and .h											|
																	|
----------------------------------------------------------------------------------------------------------------------------------------


Guide for execution:
----------------------------------------------------------------------------------------------------------------------------------------
																       	|
- If possible, opet whole project as an existing project( if it can be done with your IDE)					       	|
																	|
- If not, project is done in code::block, so clicking on a file "Airline reservation system.cbp" will be enough				|
																	|
- After opening the project, compile it and it will be done as .dat, .h and .c are not in separate folders, 				|
	for this reason to make it easier to compile											|
																	|
- If more problems occur, create new project with your IDE and add existing files from "src" folder and compile.			|
																	|
----------------------------------------------------------------------------------------------------------------------------------------


Installation gude:
----------------------------------------------------------------------------------------------------------------------------------------
																	|
1. Folder "exe" must be moved to hard disk because data files will be created and updated during the run of the program. 		|
2. Once it is done, run program by clicking on "Airline reservation system.exe" and choose desired action.				|
																	|
NOTE: library "libgcc_s_dw2-1.dll" is included in "exe" folder and must be copied to C:\Windows\System32 if there is not already one.	|
																	|
